Understanding the Box Model

This page demonstrates how CSS styles and the box model work together to create visual layouts. Each element on this page is a rectangular box with content, padding, borders, and margins.

Content Box

The inner part of the box where your text and images appear. Its dimensions can be set with width and height properties.

Padding

Clears an area around the content. The padding is transparent and affects the background of the element.

Border

Goes around the padding and content. You can style it with different colors, widths, and styles.

Margin

Clears an area outside the border. The margin is completely transparent and creates space between elements.
